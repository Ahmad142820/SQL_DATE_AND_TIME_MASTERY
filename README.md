# 📊 Mastering SQL Date & Time Functions
A dedicated reference guide demonstrating intermediate SQL capabilities for handling temporal data. This project showcases 10 foundational date and time manipulation functions essential for data profiling, feature engineering, and time-series analysis.


## 🚀 Functions Covered

 #  Function / Keyword / Purpose 
  1  `NOW()`  Retrieves the current system date and time. 
 2  `CURRENT_DATE()`  Extracts today's date only. 
 3  `EXTRACT()`  Pulls specific components (Year, Month, Day) from a timestamp. 
 4  `AGE()`  Calculates the exact time interval between a past date and today.
 5  `TO_CHAR()`  Formats dates into custom, human-readable strings (e.g., `DD-Mon-YYYY`). 
 6  `DATE_PART()`  Isolates specific sub-fields, such as the day of the week. 
 7  `DATE_TRUNC()`  Rounds a timestamp down to a specific precision (e.g., start of the month). 
 8  `INTERVAL`  Performs date arithmetic (e.g., adding or subtracting 6 months). 
 9  `CURRENT_TIME()`  Retrieves the current system time only. 
 10  `TO_DATE()`  Converts text strings into standard database date formats.
